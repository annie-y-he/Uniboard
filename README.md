## UNIBOARD
A 25-key keyboard that can type all unicode characters that exist or will ever exist.
### Usage
```
Make sure python 'keyboard' module is installed, use 'pip install keyboard'
```
```
To use the keyboard, run 'py uniboard.py'
```
```
To exit and save to 'out.txt', press 'esc'
```
```
To find character mappings, search in 'mapping.txt'
```
```
To generate mapping files, do 'py gen.py'
```
### Keyboard layout

```
q w e r   u i o p
a s d f   j k l ;
z x c v   m , . /
    s-p-a-c-e    
```
